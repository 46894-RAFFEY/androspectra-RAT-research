## 🔒 Source Code Policy

**Why no source code is included in this public repository:**

1. **Legal Risk**: Distributing functional malware violates:
   - Pakistan PECA 2016 (Prevention of Electronic Crimes Act)
   - US Computer Fraud and Abuse Act (CFAA)
   - UK Computer Misuse Act 1990
   - EU Computer Crime Directives
   - **GitHub Terms of Service** (Explicitly prohibits malware distribution)

2. **Employment Risk**: Public distribution of malware code could:
   - Trigger automatic rejection by security-conscious employers
   - Result in background check failures
   - Violate confidentiality agreements with previous employers
   - Damage professional reputation with "security researchers"

3. **Criminal Liability**: The creator/distributor can be prosecuted even if:
   - Code is "educational"
   - Labeled "for research only"
   - Never directly used to harm anyone
   - Source code doesn't execute on its own

4. **Responsible Disclosure**: Security researchers follow:
   - **Private repositories** for functional code
   - **Institutional code review** before any sharing
   - **Verified recipients only** (not public GitHub)
   - **Delayed publication** after vendor patches
